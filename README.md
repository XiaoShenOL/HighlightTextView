# HighlightTextView
对SpannableStringBuilder进行封装，轻松实现部分文字高亮、点击、图片替换等功能。
# 添加依赖
- 方法1：
~~~
  allprojects {
      repositories {
        ...
        maven { url 'https://jitpack.io' }
      }
  }
  
  dependencies {
      compile 'com.github.albert-lii:HighlightTextView:1.0.0'
  }
~~~
- 方法2：
~~~
  compile project(':highlighttextview')
~~~
